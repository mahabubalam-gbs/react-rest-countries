# REST Countries Explorer

# A Modern React App to Explore Countries Worldwide

🔗 Live Website:
[React-rest-countries](https://exquisite-marigold-ba127d.netlify.app/)

# Installation

Clone the repository
```
git clone https://github.com/mahabubalam-gbs/react-rest-countries.git
```
Navigate to project folder
```
cd rest-countries-react
```
Install dependencies
```
npm install
```
Run the app
```
npm run dev
```
📡 API Endpoint

🔹 Get All Countries

```
https://restcountries.com/v3.1/all?fields=name,capital,currencies,flags,region,population,cca3,ccn3,cca2
```

💡 How It Works

Fetches data from REST Countries API

Stores data using React state

Dynamically renders country cards

Uses reusable components for clean structure
